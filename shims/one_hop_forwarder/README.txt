To run the one hop detour forwarder through repy, run the command:
---------------------------------------------------------------------------------------------------------
   $> python repy.py restrictions.one_hop_forwarder dylink.repy one_hop_forwarder.repy TCP_PORT UDP_PORT
---------------------------------------------------------------------------------------------------------


If you would like to run the nat_forwarder in python, without any restrictions,
then run the command:
---------------------------------------------------------------------------
   $> python run_unrestricted_onehopdetour_forwarder.py TCP_PORT UDP_PORT
---------------------------------------------------------------------------
