To run the nat forwarder through repy, run the command:
------------------------------------------------------------------------------------
   $> python repy.py restrictions.nat_forwarder dylink.repy nat_forwarder TCP_PORT
------------------------------------------------------------------------------------


If you would like to run the nat_forwarder in python, without any restrictions,
then run the command:
---------------------------------------------------------
   $> python run_unrestricted_nat_forwarder.py TCP_PORT
---------------------------------------------------------
